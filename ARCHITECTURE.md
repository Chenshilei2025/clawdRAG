## 数据模型

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│                             DATA MODELS                                                  │
└─────────────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────────────┐
│  InboundMessage (入站消息)                                                               │
│  ├─ channel: str           (渠道标识: cli, api, slack, etc.)                           │
│  ├─ sender_id: str         (发送者ID)                                                    │
│  ├─ chat_id: str           (会话ID)                                                      │
│  ├─ content: str           (消息内容)                                                    │
│  ├─ media: List[Media]     (多媒体附件)                                                  │
│  │  └─ type: str           (image, video, audio, document)                              │
│  │  └─ url/path: str       (媒体URL或本地路径)                                          │
│  └─ metadata: dict         (元数据: timestamp, message_id, etc.)                        │
└─────────────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────────────┐
│  OutboundMessage (出站消息)                                                              │
│  ├─ channel: str           (目标渠道)                                                    │
│  ├─ chat_id: str           (目标会话)                                                    │
│  ├─ content: str           (响应内容)                                                    │
│  ├─ media: List[Media]     (响应附件)                                                    │
│  └─ metadata: dict         (元数据: progress, tool_hint, etc.)                          │
└─────────────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────────────┐
│  ToolResult (工具执行结果)                                                               │
│  ├─ success: bool           (是否成功)                                                   │
│  ├─ data: Any              (返回数据)                                                    │
│  ├─ error: str | None      (错误信息)                                                    │
│  └─ metadata: dict         (执行元数据: duration, tokens_used, etc.)                    │
└─────────────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────────────┐
│  Session (会话)                                                                         │
│  ├─ key: str                (会话键: channel:chat_id)                                    │
│  ├─ messages: List[dict]    (消息历史)                                                   │
│  ├─ last_consolidated: int  (最后归档位置)                                               │
│  ├─ created_at: datetime    (创建时间)                                                   │
│  └─ updated_at: datetime    (更新时间)                                                   │
└─────────────────────────────────────────────────────────────────────────────────────────┘
```
